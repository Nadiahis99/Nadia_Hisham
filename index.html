About

The engineer behind the code.

I'm a Software Engineer specialized in ASP.NET Core, C#, and SQL Server — currently working as a Software Engineer Intern at Sieraj and a graduate of the Digital Egypt Pioneers Initiative (DEPI) Full-Stack .NET track.

My graduation project, CVision, is an AI-powered recruitment platform for CV analysis, AI interviews, and applicant tracking, built on ASP.NET Core and Entity Framework Core following Clean Architecture principles.

As a freelancer, I help businesses move fast — from scoping and architecture through deployment — delivering clean, maintainable, and scalable code for clients across the UAE and beyond.

AI tools are part of how I build, not a shortcut around understanding. I use Claude, Cursor, and GitHub Copilot daily to prototype faster, refactor safely, and stay focused on architecture while the tooling handles the boilerplate.
